# Librería Getnet JavaScript

Este directorio debe contener la librería oficial de Getnet:

- `getnet.min.js` - Librería JavaScript oficial de Getnet según "Documentacion Javascript 1.0.pdf"

## Instrucciones

1. Obtener la librería oficial `getnet.min.js` de Getnet
2. Colocarla en este directorio: `/app/static/getnet/getnet.min.js`
3. El módulo `getnet_linux.js` la importará automáticamente

## Nota

La librería debe ser compatible con Web Serial API para funcionar en Linux/Chromium.
